# MaviKapi Admin Kurulum

1. `config/database.php` dosyasinda veritabani bilgilerini guncelleyin.
2. `properties` adinda bir tablo olusturun.
3. API test icin `api/properties.php` endpointini calistirin.
4. Admin paneli icin `admin/index.html` dosyasini tarayicida acin.

Not: Bu klasor yapsi, Vite tabanli React arayuzune ek olarak sunucu tarafi entegrasyon ihtiyaci icin hazirlanmistir.
